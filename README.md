# Fuzz

**中文** | [English](./README_EN.md)

一款使用[Carpet Mod](https://github.com/gnembon/fabric-carpet)操作逻辑的客户端模组，提供了一些实用功能。

默认情况下，所有功能处于禁用状态。

## 快速入门

- 主命令

  - /fuzz
    - /fuzz list

      - /fuzz list <分类标签>

    - /fuzz <规则> <选项>

  /fuzz: 查看当前已启用的规则列表

  /fuzz list: 查看完整规则列表

  /fuzz list <分类标签>: 查看指定分类标签下的规则列表

  /fuzz <规则> <选项>: 修改指定规则的选项
